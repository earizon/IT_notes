What's new in version 1.1?

  • Print to paper support:
    A printed-book-style is now available, useful for first-lectures (start-to-end) approach
    Now you can look at info like a map or like a book with Gutenberg seal-of-approval. To create a printed book just use your browser "Print..." menu.
    Or you can also filter out non-interesting content to create sub-books first.
    Very wide schemas will break, anything else will (hopefully) work.
  • Cleaned Search Interface and keyboard shortcuts now in place.
    Press key "S" to access the search menu (or "/" for vim supremacists)
    Also the Search menu can be open automatically with pre-filled search dialog with pre-selected topics and/or free text.
    Contact with HelpMan for examples.
  • smoother zoom
  • and automatic background coloring of divs, offering a visual help while zoomin in/out
