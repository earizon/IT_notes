HelpMan to the rescue!!!

Basic Ussage

  double-click (mouse) or long-press (touch-screen) on a cell to zoom into highlighted content

Searching for content

  1. Click on the "telescopic sight" icon to open the search/filter menu.
    Alternatively you can press the keyboard shortcuts "S" or "/".
  2. Search for matching text in any zoomable element by filling the regex text input.
    The "single-line" checkbox can be cheked to search only matching text in a single-line and the "case-match" checkbox to match only if Upper/lower case match.
    NOTE: user-input is interpreted as a regular expresion.
    Advanced users can profit from this feature to create powerful queries
    Normal users can ignore this feature or, if curious, learn about the magic of regular expression by clicking [here].
  3. Optionally click on any topic to filter out/in any non/related topics. The AND/OR checkbox let's to choose whether content must contain ALL or ANY selected topic.
    Only zoomable contant that has been previously tagged or labeled with matching topics will be searched for.
  4. Finally click on the "Search Now!" button (or Press "Enter" key) to execture the search
  5. Matching zoomable content will be highlighted while non-matching zoomable elements will be hidden by default.
  6. Browse visually around the highlighted content to see if the context matches with the search.
  7. Press the "ESC" key in your keyboard or the [show all] link to show again all cells and their relation in the context of the matched content (the "neighborhood" of matching elements)
NOTE: When just a single zoomable element match the search its content will automatically be zoomed-in
This behaviour allows to create external links to you page that automatically opens the desired cell/content using a link similar to: [this example link]
Otherwise the different zoomable elements will be highlighted and the user will choose which one to zoom-in.
Once you click on one of the the left/right arrows icons can be clicked to move to previous/next search match, or you can close the zoom and click on another zoomable element.

Sharing content

  • The full page book can be shared as a normal web page by sharing its URL (Universal Resource Locator)
  • If you want to put the attention or highlight some particular topic inside the book,add the query parameter "topics=topic1,topic2,..". For example:
    Share the linux administration page highlighting topics related to audit_user:
    ./DevOps/linux_administration_summary.html?topics=audit_user
    Share the same page, highlight other topic (learn_x_in_y_min):
    ./DevOps/linux_administration_summary.html?topics=learn_x_in_y_min
  • The URL query parameter can be used to share a text-search. For example:
    Share the linux administration page highlighting any cell including the word "disk": ./DevOps/linux_administration_summary.html?query=disk
    Or containing the word "disk" but restricting to the topic audit_user: ./DevOps/linux_administration_summary.html?query=disk&topics=audit_user
  • If you want to show the search Menu add the URL parameter showSearchMenu. For Example:
    ./DevOps/linux_administration_summary.html?showSearchMenu
    or a search with text and topics pre-selected
    ./DevOps/linux_administration_summary.html?showSearchMenu&query=ssh&topics=audit_user

Editing the content

  • Content is standard html
  • Use GitHub pull request to request occasional changes.
  • Become a member of The Single-Page-Book Project@GitHub to add you own IT_notes page books or publish in your prefered repository
  • Random UUID can be generated (amongst others) by: clicking here

    The GENERATED UUID can be placed "anywhere" and used to create safe links to concrete "cells" that will work with regular expression search.
    This allow to create and share inmutable URLs similar to http://..../map.html?query=GENERATED_UUID pointing to some cell or region of cells inside the map.
